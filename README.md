# ProtectIT

Сайт мини-игр с тематикой защиты от DDoS-атак.
Кейс от DDoS-Guard "DDoS-ниндзя" хакатон весна 2026 ДГТУ.

# О проекте

ProtectIT — веб-игра, в которой игрок выступает в роли инженера по безопасности, отражающего волны DDoS-атак на сервер. Цель — продержаться как можно дольше, не допустив падения сервера.

# Особенности

1. Три типа защиты: WAF, Rate Limiter, CAPTCHA. </br>
2. Написано на Flask с использованием SQLite. </br>
3. Корги ૮₍ • ᴥ • ₎ა

# Стек

1. Python 3.9+
2. Flask
3. SQLite
4. HTML / CSS / JavaScript

# Структура проекта

protectit/
├── app.py - Главный файл. Мозг и сердце нашей защиты</br>
├── requirements.txt - Список всего, что нужно подтянуть (как подкрепление)</br>
├── database/ - Тут лежат наши записи о героях и их подвигах</br>
├── static/ - CSS стили, JS скрипты и картинки для интерфейса</br>
│   ├── css/ - Вся красота интерфейса</br>
│   └── js/ - Фронтенд-логика игры (ваши клики превращаются в победы)</br>
└── templates/ - HTML-шаблоны страниц</br>
    ├── index.html - Главная арена битвы</br>
    └── leaderboard.html - Доска почета</br>
